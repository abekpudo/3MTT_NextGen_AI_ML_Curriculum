Vision Capstone Design

Overview
Guide the design of a computer vision capstone with clear objectives, dataset selection, evaluation criteria, and deployment considerations.

Why This Topic Matters
A vision capstone should balance technical novelty with a robust plan for data, model selection, evaluation, and delivery.

Lecture Goals
- Define the target problem and success metrics
- Choose a dataset and annotation strategy
- Plan model architecture and training workflow
- Scope deployment and user interaction
- Prepare reporting and demo materials

Key Concepts
- problem scoping
- dataset design
- model planning
- deployment scope
- presentation

Classroom Example / Case Study
- Outline the capstone problem and the value it will deliver
- Map the project into data collection, modeling, and deployment phases
- Set milestones for prototype, refinement, and final demo

Practical Implementation Notes
- Keep the project scope manageable.
- Document the full pipeline clearly.
- Align the capstone with real user needs and evaluation criteria.
