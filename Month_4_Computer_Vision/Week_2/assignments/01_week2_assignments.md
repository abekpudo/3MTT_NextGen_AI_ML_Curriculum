 Week2 Assignments

 Objective

- Develop a computer vision solution using CNNs or transfer learning.
- Prepare the dataset, train a model, and evaluate predictions.
- Document how the model would be deployed in production.

 Requirements

1. Image preprocessing and training code
2. Evaluation metrics and visualizations
3. Deployment planning or demonstration
4. Project summary and improvement ideas

 Evaluation Criteria

| Item | Weight |
|------|--------|
| Image preprocessing and training code | 25% |
| Evaluation metrics and visualizations | 25% |
| Deployment planning or demonstration | 25% |
| Project summary and improvement ideas | 25% |
