CNN Architecture

Overview
Describe the architecture of convolutional neural networks and the role of convolution, pooling, and activation layers in image understanding.

Why This Topic Matters
CNNs are the standard deep learning architecture for image tasks because they exploit spatial structure while sharing weights and reducing parameters.

Lecture Goals
- Explain convolutional layers and learned filters
- Describe pooling and feature hierarchy
- Review activation functions and normalization
- Compare shallow and deep CNNs
- Discuss overfitting and regularization in vision models

Key Concepts
- convolution
- feature maps
- pooling
- activation
- batch normalization

Classroom Example / Case Study
- Illustrate a simple CNN architecture from input image to flattened features
- Explain how convolutional filters detect edges, textures, and objects
- Discuss the effect of receptive field size on feature extraction

Practical Implementation Notes
- Use small kernels and multiple layers instead of a large single filter.
- Batch normalization often improves training stability in CNNs.
- Monitor for overfitting with validation loss and augmentations.
