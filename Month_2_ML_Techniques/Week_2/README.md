 Week 2: Supervised Learning & Model Building

Duration: 5 days | Theme: Classification & Regression | Difficulty: Intermediate

---

 🎯 Learning Objectives

By the end of this week, you will:
- ✅ Build and train supervised learning models
- ✅ Master key classification algorithms
- ✅ Build regression models
- ✅ Evaluate models with appropriate metrics
- ✅ Handle class imbalance
- ✅ Create sklearn pipelines
- ✅ Select best performing models

---

 📖 Topics Covered

 Day 1: Regression Algorithms
- Linear Regression
- Polynomial Regression
- Logistic Regression for binary classification

 Day 2: Classification Algorithms I
- Decision Trees
- Random Forests
- K-Nearest Neighbors (KNN)

 Day 3: Classification Algorithms II
- Support Vector Machines (SVM)
- Naive Bayes
- Gradient Boosting

 Day 4: Model Evaluation
- Classification metrics (Accuracy, Precision, Recall, F1)
- Regression metrics (MAE, RMSE, R²)
- Confusion matrices and ROC curves
- Cross-validation strategies

 Day 5: Model Comparison & Selection
- Comparing multiple models
- Statistical tests for model comparison
- Feature importance analysis
