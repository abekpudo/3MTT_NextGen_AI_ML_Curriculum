 Week Exercises: Week 3: AWS Cloud Essentials | What is cloud computing? | Cloud service models (IaaS, PaaS, SaaS) | AWS architecture and global infrastructure

 Objectives
Apply the week’s concepts through practical, hands-on tasks.

1. Configure AWS CLI and authenticate with a user that follows least privilege.
2. Launch an EC2 instance and SSH into it securely.
3. Store a dataset in S3 and set an appropriate bucket policy.
4. Create an IAM role or policy that limits access to a single service.
5. Summarize the architecture and security controls used.
