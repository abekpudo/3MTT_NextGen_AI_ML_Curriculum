 Week 3: AWS Cloud Essentials

Duration: 5 days | Theme: Cloud Computing Fundamentals | Difficulty: Intermediate

---

 🎯 Learning Objectives

By the end of this week, you will:
- ✅ Understand cloud computing concepts
- ✅ Master AWS account setup and configuration
- ✅ Deploy applications on EC2
- ✅ Store files in S3
- ✅ Understand AWS IAM and security
- ✅ Deploy a simple web application

---

 📖 Topics Covered

 Day 1-2: Cloud Computing Fundamentals
- What is cloud computing?
- Cloud service models (IaaS, PaaS, SaaS)
- AWS architecture and global infrastructure
- Core AWS services overview

 Day 3: AWS EC2 & Networking
- Launch and manage EC2 instances
- Understand security groups and key pairs
- Connect via SSH
- Configure basic web server

 Day 4: Storage & Data Management
- S3 bucket creation and management
- Upload and manage files
- Understand versioning and lifecycle policies
- S3 security and access control

 Day 5: Deployment Project
- Deploy Python web application
- Set up environment on EC2
- Deploy database
- Scale and optimize
