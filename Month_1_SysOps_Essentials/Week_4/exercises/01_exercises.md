 Week Exercises: Week 4: DevOps & Automation | DevOps culture and principles | Containerization concepts | Docker basics and containers

 Objectives
Apply the week’s concepts through practical, hands-on tasks.

1. Write a Dockerfile for a simple Python application.
2. Build and run the container locally, then inspect logs.
3. Create a GitHub Actions workflow that builds and tests the app.
4. Add a health check endpoint and verify it in a browser or curl.
5. Document how the pipeline supports deployment and rollback.
