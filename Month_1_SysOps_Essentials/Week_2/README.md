 Week 2: Linux & Bash Fundamentals

Duration: 5 days | Theme: Command Line Mastery & System Administration | Difficulty: Intermediate

---

 🎯 Learning Objectives

By the end of this week, you will:
- ✅ Understand Linux operating system fundamentals
- ✅ Master Bash shell and command-line operations
- ✅ Work with file systems and permissions
- ✅ Write basic shell scripts
- ✅ Understand processes and system management
- ✅ Use advanced command-line tools

---

 📖 Topics Covered

 Day 1-2: Linux Fundamentals
- Linux history and architecture
- File system structure and navigation
- Users and groups
- File permissions and ownership
- Working with directories and files

 Day 3-4: Bash Shell Scripting
- Bash basics and scripting fundamentals
- Variables and data types
- Conditionals and loops
- Functions and modules
- Text processing with grep, sed, awk

 Day 5: System Administration
- Process management
- Package management (apt, brew, yum)
- System monitoring and logs
- Cron jobs and task scheduling
- Network basics
