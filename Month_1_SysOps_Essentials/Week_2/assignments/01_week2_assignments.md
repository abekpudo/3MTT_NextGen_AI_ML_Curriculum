 Week 2 Assignment: Linux, Bash, and System Automation

 Objectives
- Apply Linux command-line, Bash scripting, and system administration skills to solve real-world problems.
- Build and document a robust automation or system management workflow.
- Demonstrate best practices in security, error handling, and reproducibility.

 Assignment Tasks
1. Linux Command Mastery:
	- Use shell commands to explore the filesystem, manage permissions, and monitor processes.
	- Document at least five advanced commands or command combinations (e.g., `find | grep`, `awk`, `sed`, `chmod`, `chown`, `ps aux | grep`).

2. Bash Scripting:
	- Write a Bash script that automates a common admin task (e.g., backup, log analysis, user report, or scheduled cleanup).
	- The script should include argument validation, error handling, and logging.

3. System Administration:
	- Demonstrate user/group management and service control (e.g., add/remove users, set permissions, start/stop a service).
	- Document the steps and include screenshots or terminal output.

4. Automation & Scheduling:
	- Set up a cron job or systemd timer to run your script automatically.
	- Document how to verify and monitor the scheduled task.

5. Documentation:
	- Provide a `README.md` with setup instructions, script usage, and troubleshooting tips.
	- Include a diagram or flowchart of your automation workflow.

 Evaluation Criteria
| Item | Weight |
|------|--------|
| Linux command mastery & documentation | 20% |
| Bash scripting & automation | 25% |
| System administration scenario | 20% |
| Scheduling & monitoring | 15% |
| Documentation & reproducibility | 20% |
