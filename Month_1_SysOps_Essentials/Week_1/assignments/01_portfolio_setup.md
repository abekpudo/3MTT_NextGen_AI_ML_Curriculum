 Week 1 Assignment: Personal Portfolio Setup

 📋 Assignment Overview

Create a complete professional development environment and GitHub portfolio that will serve as the foundation for your 3MTT learning journey.

Weight: 100 points (Week 1 Grade)  
Due: End of Week 1  
Submission: GitHub repository link via learning platform

---

 🎯 Learning Objectives

By completing this assignment, you will:
- Set up a professional Python development environment
- Master Git version control workflows
- Create a professional GitHub portfolio
- Establish your personal brand as an ML engineer
- Practice documentation and presentation skills

 🆘 Common Issues

| Issue | Solution |
|-------|----------|
| Can't push to GitHub | Check SSH key setup |
| Virtual env not working | Reinstall Python, use python3 |
| VS Code not recognizing Python | Select interpreter (Cmd+Shift+P) |
| Jupyter won't start | Check installation: `pip install jupyter` |
| Merge conflicts | Follow exercise instructions |
| Git commit fails | Configure user.name and user.email |
