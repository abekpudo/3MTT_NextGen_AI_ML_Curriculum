 Week 1: Foundations & Program Orientation

Duration: 5 days | Theme: Getting Started & Git Fundamentals | Difficulty: Beginner

---

 🎯 Learning Objectives

By the end of this week, you will:
- ✅ Understand the 3MTT program structure and expectations
- ✅ Set up a professional development environment
- ✅ Master Git fundamentals and GitHub workflow
- ✅ Create your first GitHub repository
- ✅ Practice basic version control workflows

---

 📖 Topics Covered

 Day 1-2: Program Orientation & Environment Setup
- Program structure, timeline, and expectations
- Python installation and virtual environments
- IDE setup (VS Code)
- Package management with pip and conda
- Creating a GitHub account

 Day 3-4: Git Fundamentals
- Version control concepts and workflow
- Git basics: init, add, commit, log
- Understanding the staging area
- Branching basics
- Git configuration and best practices

 Day 5: GitHub & First Repository
- GitHub account and repository creation
- Cloning and forking repositories
- Creating a professional README.md
- First commit and push to GitHub
- Understanding .gitignore
