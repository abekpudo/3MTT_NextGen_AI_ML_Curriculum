Week 1 Cheatsheet: Terminal & Environment Setup

Key Concepts
- Shell basics (bash, zsh)
- Navigation and file management
- Environment variables and virtual environments
- CLI productivity (aliases, history, tab completion)
- Integrating Git and Python in the terminal

Essential Commands
- `pwd` — Print working directory
- `ls -la` — List files (detailed)
- `cd <dir>` — Change directory
- `mkdir <dir>` — Create directory
- `cp <src> <dst>` — Copy files
- `mv <src> <dst>` — Move/rename files
- `rm <file>` — Remove file
- `cat <file>` — View file contents
- `code .` — Open VS Code in current directory
- `python3 -m venv venv` — Create virtual environment
- `source venv/bin/activate` — Activate virtual environment
- `pip install <package>` — Install Python package
- `git <command>` — Use Git from the terminal

Productivity Tips
- Use tab completion to speed up typing
- Use `history` and `!<number>` to repeat commands
- Create aliases for frequent commands (e.g., `alias gs='git status'`)
- Use `clear` to clean up your terminal

Troubleshooting
- Check your PATH: `echo $PATH`
- Find command location: `which <command>`
- View hidden files: `ls -a`
