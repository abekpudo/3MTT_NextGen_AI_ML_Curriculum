 Month 3 Capstone Project: Specialized NLP Solution

 Objectives
- Deliver a production-grade NLP application that solves a linguistic or informational problem.
- Master the pipeline from web scraping/data collection to Transformer fine-tuning.
- Showcase a live, interactive interface for your model.

 Assignment Tasks
1. Data Provenance:
	- Provide a script or notebook showing how you collected your data (Scraping, API, or Custom Dataset).
	- Include a Data Cleaning Log showing the transformation from raw text to model-ready tokens.

2. The "Specialist" Model:
	- Fine-tune a state-of-the-art Transformer (BERT, RoBERTa, T5, etc.) for your specific task.
	- Comparative Baseline: You must show that your Transformer model significantly outperforms a traditional baseline (e.g., Naive Bayes).

3. In-Depth Evaluation:
	- Provide standard metrics (F1, Precision, Recall).
	- Provide a Task-Specific Metric (e.g., BLEU for translation, ROUGE for summary, or MRR for retrieval).
	- Include a visualization of the most important tokens (e.g., SHAP values or Integrated Gradients).

4. Interactive Deployment:
	- Deploy a Gradio or Streamlit interface.
	- Your demo should allow a user to type a sentence and see the model's prediction and confidence score instantly.

5. Final Technical Paper:
	- A 2-3 page Markdown report covering:
		- Problem Scoping.
		- Data Challenges.
		- Model Selection & Hyperparameters.
		- Deployment & Future Work.

 Submission Requirements
- Link to your GitHub/GitLab repository.
- Link to the hosted demo (e.g., Hugging Face Spaces).
- The final report in PDF or Markdown format.

 Evaluation Criteria
| Item | Weight |
|------|--------|
| Data Collection & Preprocessing Quality | 30% |
| Model Performance vs Baseline | 30% |
| Interactive UI & User Experience | 20% |
| Documentation & Reporting Depth | 20% |
